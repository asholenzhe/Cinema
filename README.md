# Cinema

Минимальное SPA на React + TypeScript для просмотра фильмов через API poiskkino.dev.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Создать `.env` из `.env.default` и добавить API ключ:

```env
VITE_POISKKINO_API_KEY=ВАШ_КЛЮЧ
```

3. Запустить:

```bash
npm run dev
```

## Что реализовано

- Список фильмов с бесконечной подгрузкой по 50 элементов (`/v1.5/movie`, cursor pagination).
- Фильтры по жанрам, рейтингу и году.
- Синхронизация фильтров в URL через search params.
- Поиск по названию (`/v1.4/movie/search`).
- Страница деталей фильма (`/v1.4/movie/{id}`).
- Избранное с сохранением в `localStorage`.
- Подтверждение добавления в избранное через модальное окно.
- Режим сравнения до 2 фильмов (при выборе третьего первый сбрасывается).
